# How to Start the Frontend Server

## Correct Commands:

1. **Navigate to the frontend directory:**
   ```powershell
   cd "C:\MAIN PROJECT\frontend"
   ```

2. **Start React development server:**
   ```powershell
   npm run dev
   ```

   The server will start at: `http://localhost:5173/` (or similar port)

## Important Notes:
- Frontend is in `C:\MAIN PROJECT\frontend\`
- You must be IN the frontend directory to run `npm run dev`
- The frontend folder is separate from the backend folder
